\begin{abstract}
Scientific data discovery requires search that acts, not just retrieves. Data is heterogeneous in format and expression, varies in the quality and quantity of metadata, and is scattered across storage systems. A human scientist might compensate naturally---converting units mentally, inferring from partial clues, checking secondary filesystems---but human effort does not scale to the billions of objects produced by modern science. No retrieval pipeline replicates this reasoning, because the required operations are not just arithmetic or navigational, but inferential. In the era of agents, scientific discovery demands scalable data exploration driven by LLM agents that explore, transform, and reason over data they have never seen.

We present CLIO Search, a framework for agentic scientific discovery that implements a new harness with intelligent orchestration, science-aware operators, federated connectors, and metadata-adaptive discovery. On cross-unit queries, CLIO achieves 0.99~P@5 where baselines score 0.40, improving real-world precision by 87\% on NOAA observatory data. Wrapping CLIO behind a single tool call reduces LLM token consumption by 52\% and wall time by 91\% compared to a native agent baseline, while answering all 10~queries. Cross-unit correctness is verified at distributed scale on 70K arXiv documents across 4~NVIDIA GH200 nodes, where querying in Pa, kPa, bar, and atm returns identical results at sub-50\,ms latency. Integration with IOWarp's tiered blob storage demonstrates 0.5\% corpus inspection per query where the native API returns everything unfiltered. All code and benchmarks are open-source.
\end{abstract}
