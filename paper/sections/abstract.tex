\begin{abstract}
Scientific data discovery requires search that thinks before it acts. An effective system must profile each corpus to understand what metadata exist, decide which strategies apply, execute domain-specific reasoning such as the arithmetic to recognize that 200~kPa, 200000~Pa, and 0.2~MPa describe identical pressure, and refine when results are insufficient. This observe-decide-act-evaluate cycle distinguishes agentic search from conventional retrieval. Few current systems perform these operations in combination.

We present CLIO Search, an agentic retrieval system implementing a four-stage pipeline: Discover, Reason, Transform, and Execute. Science-aware operators run as deterministic database predicates alongside BM25 and dense search, with federated access across heterogeneous backends.

CLIO achieves 0.99~P@5 on cross-unit queries (BM25: 0.40); same-unit controls confirm parity (0.30--0.45). Precision improves 87\% on real NOAA data. At scale on 2.5M arXiv abstracts across 4~GH200 nodes: sub-50\,ms latency, 0.94 weak scaling. Code and benchmarks will be released.
\end{abstract}
