# 6. Conclusion

Scientific data retrieval at HPC facilities fails not because retrieval algorithms are weak, but because they are blind to the physical semantics of the data they index. When a scientist asks for pressures between 190 and 360 kPa, no existing system recognizes that "200000 Pa" in an HDF5 attribute and "0.2 MPa" in an S3-hosted dataset both satisfy the query. We addressed this gap with clio-agentic-search, a retrieval system built around science-aware operators---deterministic, pluggable primitives that perform arithmetic SI conversion, formula normalization, and federated multi-backend search as first-class retrieval branches alongside standard BM25 and dense vector pipelines.

These gains matter because AI agents are now orchestrating scientific workflows end-to-end, from hypothesis generation through experiment execution, yet their retrieval layers treat measurements as opaque strings.

Our current prototype covers 13 units across 5 SI domains, far fewer than the 531 units in CQE or the thousands in the QUDT ontology; derived and compound units remain unsupported. Measurement extraction relies on regular expressions rather than dependency parsing, which limits robustness on syntactically complex scientific prose. The composite fusion score uses hand-tuned weights rather than learned reranking via cross-encoders or late-interaction models. All experiments run on a single-node benchmark corpus, not a petabyte-scale parallel filesystem---the federated architecture is designed for scale, but we have not yet validated it at ALCF or OLCF production workloads.

Several directions follow naturally. Integrating the QUDT ontology would extend unit coverage to derived and compound units without architectural changes. Wrapping the retrieval API as a Model Context Protocol server would expose science-aware operators to any MCP-compatible agent framework. Deployment on DOE leadership-class systems---Polaris, Frontier, Aurora---will test federated search at the scale it was designed for. More broadly, the principle that domain knowledge should execute as verifiable arithmetic within the retrieval pipeline, rather than be approximated by learned representations, extends beyond SI units to chemical identifiers, genomic coordinates, and any domain where correctness is non-negotiable. If retrieval systems are to serve as the foundation for autonomous scientific agents, they must understand the data they search---not merely the text that describes it.
