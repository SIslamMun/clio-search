# Plan 03-02: Formula Normalization Pipeline + Hybrid 4-Branch Retrieval Pipeline

## Context
This plan covers formula normalization (the second science-aware operator) and the hybrid retrieval pipeline that integrates all four branches. Formula normalization is a smaller contribution than dimensional conversion but is novel in its integration with measurement operators. The hybrid pipeline description shows how science-aware operators execute as first-class parallel branches alongside standard lexical and vector retrieval.
