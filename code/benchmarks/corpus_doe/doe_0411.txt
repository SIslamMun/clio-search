This umbrella contains data from all instrumentation associated with the aircraft’s navigational location, motion, and attitude data. Navigational instrumentation on the Gulfstream-159 includes navigational data recorded from the AIMMS-20 (for other AIMMS produced data see the instrument category MET-AIR), CMigits global positioning system (GPS) & Inertial Navigational System (INS) Tactical System, Differential Survey Module (DSM), and VectorNav VN-200 INS. For a higher-order product that contains navigational data integrated with meteorological data and sampling masks/flags to aid analysis please see the categorical value added product NAVMET-AIR.
